# プロ野球ベスト9作成アプリ(Flask/ポートフォリオ)
選手をポジションごとに選択し、自分の考えたベスト9を作成できる趣味アプリです。
***
## 作成目的
Python学習の為、自分の好きな野球をテーマに制作。DjangoではなくFlaskを選択した理由はプログラミングの基礎理解を優先し、どのように動いているのかじっくりと理解しながら学習したいと考えたため。
***
## 使用技術
- Python
- Flask
- SQLite
- HTML/CSS
***
## 主な機能
### メイン機能
- 選手一覧、ポジションごとの選手一覧、チームごとの選手一覧の表示
- ポジションごとに選手を選択しベスト9を作成、削除
- ユーザーごとに作成されたベスト9の一覧を表示 
### 認証機能
- サインアップ、ログイン機能
- サインアップ、ログインフォームのバリデーション
  ‐ 未入力、文字数制約、ユーザー名の重複禁止
### その他
‐ エラーハンドリング登録
‐ csvファイルより選手データ取り込み
***
## 工夫した点
‐ アプリファクトリーパターンを使用し、Blueprint機能と合わせて、役割を分担。追加の機能実装時にも目的別にBlueprintを分け対応しやすく設計
‐ extensionsファイルからインスタンスをインポートすることで循環インポートしにくい設計
‐ __repr__メソッドを使用しログからオブジェクトを把握できるように
## 改善点
‐ 完成を目的とした為、フロントサイドは必要最低限のものにしてあり、もっとリッチな見た目にできた点
‐ 当初目的では動的な保存画面を作成する予定でしたのでjavascriptを学習し反映させたい
‐ 登録データ数やテーブルカラム(選手成績、打率、HR数など)を増やし、歴代選手からのベスト9を作成できるようにしてみたい
