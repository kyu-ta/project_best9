# プロ野球ベスト9作成アプリ(Flask/ポートフォリオ)
プロ野球選手をポジションごとに選択し、オリジナルのベスト9を作成、保存できるWebアプリケーションです。
***
## 作成目的
PythonおよびWebアプリケーション開発の学習を目的とし、自分の好きな野球をテーマに制作しました。
DjangoではなくFlaskを選択した理由はプログラミングの基礎理解を優先し、アプリの構成や処理の流れを把握しながら実装するためです。
***
## 使用技術
- Python
- Flask
- SQLite
- HTML/CSS
***
## 主な機能
### ベスト9作成機能
- 選手一覧の表示
- ポジション別の選手一覧の表示
- チーム別の選手一覧の表示
- ポジションごとに選手を選択しベスト9を作成
- 作成したベスト9の保存、削除
- ユーザーごとに作成されたベスト9の一覧を表示
### 認証機能
- サインアップ機能
- ログイン機能
- フォームバリデーション
  ‐ 未入力チェック
  - 文字数制約
  - ユーザー名の重複禁止
### その他
- エラーハンドリング登録
- CSVファイルより選手データ取り込み
***
## 工夫した点
- アプリファクトリーパターンを使用しアプリ生成処理を分離しました。
- Blueprintを利用して、メイン機能と認証機能を分割し、役割ごとに整理しました。
- extensionsファイルから拡張機能のインスタンスを管理することで循環インポートしにくい構成を意識しました。
- __repr__メソッドを使用しデバッグ時にログからオブジェクトを把握できるようにしました。
## 改善点
- フロントエンドの見た目を改善し、UIを見やすくする。
- JavaScriptを学習し動的な操作のできる保存画面を実装する。
- 登録データ数やテーブルカラム(選手成績、打率、HR数など)を増やしより幅広いベスト9作成に対応する。

## 学んだこと
- Flaskを用いたWebアプリケーションの基本構成
- SQLAlchemyを用いたモデル定義とデータベース作成
- ログイン機能、CRUD機能、バリデーション機能、エラーハンドリング機能
- アプリ構成を意識したファイルの分割
- エラーを調査しながら実装を進める開発プロセス
